# SIMPLE CRUD API

Simple API using Nestjs with GraphQL and MongoDB.

## Stack

* Node.js 16.0.0
* Nestjs 8.0.0
* GraphQL: 15.5.1
* Mongoose 5.13.5


## Installation

```bash
$ npm install
```

## Configuration

```
After Install please modify in the .env file the Mongo url you want to use.

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
